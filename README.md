# College-ERP-Attendance-System

## Project Overview
The College ERP Attendance System is a role-based attendance management platform developed for colleges to simplify and secure attendance tracking. The system allows teachers and students to log in separately, create attendance sessions, mark attendance securely, and monitor attendance records in real time.

This project was developed as a mini project and focuses on reducing proxy attendance while giving teachers better attendance management and analytics.

---

## Features

### Teacher Module
- Secure teacher login using email and password
- Create attendance sessions
- Generate unique attendance IDs
- Share attendance PIN with students
- View live attendance updates
- Monitor individual student attendance
- Search and filter student records
- Download attendance reports
- Access attendance charts and analytics

### Student Module
- Secure login using student ID
- Receive attendance session IDs
- Enter attendance PIN
- Mark attendance
- View attendance records
- Track attendance percentage

---

## Main Security Features
- Role-based authentication
- Unique attendance session IDs
- 4-digit PIN verification
- Anti-proxy attendance mechanism
- Course-specific attendance management
